# Track It

a habit management app

## Features

- sing up
- login
- add a new habit
- remove a habit
- list of all created habits
- habit schedule

## Tecnologies

- html5
- css3
- javascript
- react
- contextAPI

## Running locally

1. clone this repository
2. install dependencies `npm i`
3. run the app with `npm start`
4. access http://localhost:3000
